🏛️ ClearGov – Privacy Policy

Last updated: March 2026

The short version: ClearGov reads form field labels and page context to generate explanations. It never reads what you type into fields. Your API key stays on your device. No data is collected, sold, or shared by ClearGov itself.

What ClearGov Does

ClearGov is a Chrome browser extension that explains government form fields in plain English. When you click on or interact with a form field on a .gov or .edu website, ClearGov reads contextual information about that field and sends it to the Anthropic AI API to generate a plain-English explanation.

What Data ClearGov Collects and Sends

When you interact with a form field, ClearGov may send the following to the Anthropic API:

  • The field's label or question text (e.g. "Date of Birth", "Adjusted Gross Income")
  • Helper text or instructions visible near the field on the page
  • The page title and URL of the government website you are visiting
  • For dropdown menus: the list of available options
  • For agreement/consent checkboxes: the visible legal text on the page

ClearGov does NOT collect or send:

  • Any values you type into form fields (your name, SSN, income amounts, dates of birth, etc.)
  • Passwords or authentication credentials
  • Any information from pages outside of .gov and .edu websites
  • Browsing history
  • Any personally identifiable information

Your Anthropic API Key

ClearGov requires you to provide your own Anthropic API key to function. This key is stored locally on your device using Chrome's built-in storage and is never transmitted to ClearGov or any server other than Anthropic's API directly. You can remove your key at any time using the "Change API key" option in the extension.

Third-Party Services: Anthropic

ClearGov uses the Anthropic API to generate explanations. Field context data described above is sent directly from your browser to Anthropic's servers. ClearGov does not have access to this data and does not store or log any API requests or responses.

Anthropic's data handling practices are governed by their own privacy policy, which you can review at anthropic.com/privacy.

Data Storage

ClearGov stores only two things locally on your device using Chrome's storage API:

  • Your Anthropic API key
  • The most recently detected form field context (used to restore state when the sidebar opens)

Neither of these is transmitted to ClearGov or any server other than Anthropic's API.

Permissions

ClearGov requests the following Chrome permissions:

  • activeTab / scripting: To read form field labels and page context on the current tab
  • sidePanel: To display the explanation panel alongside the page
  • storage: To save your API key and last field state locally
  • tabs: To detect when you navigate to a .gov or .edu site and enable the panel

Children's Privacy

ClearGov is not directed at children under 13 and does not knowingly collect any information from children. The extension is designed to assist adults filling out government forms.

Changes to This Policy

If this privacy policy changes in a material way, the "Last updated" date above will be revised. Continued use of the extension after changes constitutes acceptance of the updated policy.

Contact

If you have questions about this privacy policy, please contact us through the Chrome Web Store listing page.

ClearGov is an independent extension and is not affiliated with any government agency.
